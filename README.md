# T-UI

> A High Quantity UI Toolkit By VUE.

## usage 
```$bash
#not suuported npm yet, just foking and expanding first
#it will be
npm install tui --save
```

### import full TUI
```javascript
import Vue from 'vue';
import TUI from 'tui';
Vue.use(TUI);
```

### import part of TUI
```javascript
import Vue from 'vue';
import {Button} from 'tui';
Vue.use(Button);
```
## installation

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
